Página de Ajuda - Instruções de Uso

  • Login: Para aceder ao sistema, utilize as credenciais de administrador fornecidas.
  • Alterar Senha: No canto superior direito, clique no seu nome de utilizador e selecione 'Alterar Senha' para definir uma nova senha.
  • Sair: Para sair do sistema de forma segura, utilize a opção 'Sair' no mesmo menu.

  • Como Registar: Na página 'Empréstimos', clique em 'Adicionar Empréstimo', selecione o colaborador, o equipamento e a quantidade, e clique em 'Salvar'.
  • Como Devolver: Para registar uma devolução, encontre o empréstimo na lista e clique no botão 'Devolver'.

O processo é similar: aceda à página correspondente, utilize o botão 'Adicionar' para criar novos registos e os botões 'Editar' ou 'Excluir' para gerir os existentes.
