Gestão EPI Smart

Descrição

Gestão EPI Smart é uma aplicação web desenvolvida em Python com o framework Django, criada para gerir o empréstimo de Equipamentos de Proteção Individual (EPIs) em ambientes de trabalho, como na construção civil. O sistema permite um controlo detalhado de colaboradores, equipamentos e dos empréstimos realizados, garantindo a segurança e a conformidade com as normas de trabalho.

Este projeto foi desenvolvido como parte da avaliação do curso de Desenvolvimento de Sistemas.

Funcionalidades Principais
  • Gestão de Colaboradores: CRUD completo para o registo de funcionários.
  • Gestão de Equipamentos: CRUD completo para EPIs, com controlo de stock (quantidade total e disponível).
  • Controlo de Empréstimos: Registo de retirada e devolução de equipamentos por colaborador.
  • Controlo de Stock Automatizado: Stock atualizado automaticamente em empréstimos e devoluções.
  • Sistema de Autenticação: Login protegido com página para alteração de credenciais.
Tecnologias Utilizadas
  • Backend: Python, Django
  • Frontend: HTML, CSS, Bootstrap 5, JavaScript
  • Base de Dados: SQLite 3
Como Executar o Projeto Localmente
  1. Clone o repositório e aceda à pasta do projeto.
  2. Crie e ative um ambiente virtual (ex.: python -m venv .venv).
  3. Instale as dependências: pip install -r requirements.txt.
  4. Execute as migrações: python manage.py migrate.
  5. Crie um superutilizador: python manage.py createsuperuser.
  6. Inicie o servidor de desenvolvimento: python manage.py runserver e aceda a http://127.0.0.1:8000/.
